# How It Works

## The Model IS the Code

**Design → Execute → Observe → Modify → Redeploy**

1. **Design** — Drag processing blocks onto a visual canvas, connect them
2. **Execute** — The platform compiles the model into a live service instantly
3. **Observe** — Real-time metrics, logs, and tracing built into every block
4. **Modify** — Change any block, reconnect, adjust parameters
5. **Redeploy** — Hot deploy in seconds, instant rollback to any version

> No translation between design and code. The model IS the running service.
